# Tech Lead Skills

## Specialized Agents (Tools)

### coder_agent
An expert Software Engineer that writes clean, efficient, and well-documented Python code.
- **Inputs:** Feature requirements or code change requests.
- **Outputs:** Full Python code snippets.

### reviewer_agent
A Senior Security and Performance Reviewer that analyzes Python code for bugs, vulnerabilities, and performance.
- **Inputs:** Python code to be reviewed.
- **Outputs:** Constructive feedback, suggested improvements, or approval.

### test_agent
A Quality Assurance (QA) Automation Engineer that writes comprehensive unit tests using the `pytest` framework.
- **Inputs:** Python code to be tested.
- **Outputs:** Full `pytest` code snippets with high coverage.
